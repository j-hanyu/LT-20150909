|（パイプ）
左側のコマンドの出力結果を右側のコマンドの入力に渡す
{}（中括弧）
中括弧内で実行されたコマンドをまとめて出力する

sshコマンド

別のサーバ上でコマンドを実行する 

ssh production_web01 cat /var/www/nexlink2/current/logs/production.log

sedコマンド

入力された各行に対して変換などを掛ける 

sed -e "s/^\([0-9]*\):\[\([0-9 :-]*\)\([0-9.]*\)\] /[\2] \3 [web01:\1] /"

egrepコマンド

正規表現を使用したgrep 

egrep '(\] Started |\] Processing |\][ ]+Parameters:|\] Completed |\[FATAL\]|\[ERROR\])'

sortコマンド

入力された行をソートする 

sort -k 1,3

cutコマンド

入力された行のいらない部分を除去する 

cut -d ' ' -f 1,2,4,7,8,9,11-

結局のところ何をやっているか

各webサーバのproduction.logに対して下記の加工を実施

  • `[`で始まっていない行は前の行に結合
    （その際改行を`[NR]`に置き換える）
  • 各行の先頭に行番号を付与
  • 不要な行を除外
  • サーバ名を付与と行番号の移動

結局のところ何をやっているか

全サーバのログを結合して下記を実施

  • 日時、行番号でソート
  • 不要な部分の除去
  • [NR]を改行に戻す
